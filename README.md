# Taif Cyber Portfolio

A personal cybersecurity portfolio built using Python Flask.

This project was created to practice web development, Linux and cybersecurity concepts while building real hands-on projects.

## Features

- Password Strength Checker
- Phishing Email Analyzer
- Resume Download
- Responsive Cyber UI
- GitHub Integration
- Flask Web Application

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Git & GitHub
- Kali Linux
